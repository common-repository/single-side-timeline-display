== Installation ==

Download " single-side-timeline-display.zip" to your computer.
Login WordPress as administrator
Click on Plugins --> Add New
On "Install Plugins" page, click on "Upload"

Browse and select " single-side-timeline-display.zip" and click on "Install Now"

You can activate it now

= Just use the following Short-Code  to you page or post: =
[bs_single_timeline text="simple timeline display(,)simple timeline display"]

(,) - use this sign between your text. That way you can create as many timeline line you want.
